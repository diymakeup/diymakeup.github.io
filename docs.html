Welcome

Online Makeup is a javascript based virtual makeover tool for your websites. This is take advantage of HTML canvas and done virtual makeup of user provide images. Online Makeup used multiple javascript library for find face and get landmark and apply different effects.

Install

Online Makeup have a three support.js files and one kit.js. Support js files contain multiple javascript library files include jquery,fabric,chart..etc.

include style files like this on your HTML file

<link rel="stylesheet" type="text/css" href="styles/kit.css" />
<link href="https://maxcdn.bootstrapcdn.com/font-awesome/4.7.0/css/font-awesome.min.css" rel="stylesheet" >
<link rel="stylesheet" type="text/css" href="styles/kit.support.css" />

yes we use fontawsome for icons.

include script files like this on your HTML file

<script type="text/javascript" src="js/model.js"></script>
<script type="text/javascript" src="js/kit.support.1.js"></script>
<script type="text/javascript" src="js/kit.support.2.js"></script>
<script type="text/javascript" src="js/kit.support.3.js"></script>
<script type="text/javascript" src="js/kit.js"></script>

In kit.js contain Heart of all methods and events.

configuration

Basic initialization

<script>
    var makeoverKit = new makeoverKit({
        //configuration & options
    });
    makeoverKit.init();
</script>

Methods

1.) colorChart (which,color)

makeoverKit.colorChart('lipstick',['red','green']);
//Avaliable which is given below
//color = Array

avaliable which = lipstick,lipliner,lipgloss,eyeshadow,eyeliner,eyebrow,foundation,concealer,blush this is can change color of particular color chart.

2.) lipstick(color,amount)

makeoverKit.lipstick('red',0.5);
//color = any color
//amount = 0 to 1

This method allow to apply lipstick on image.you can also access global value window.lipstickColor or window.lipstickAmount.

3.) lipgloss(color,amount)

makeoverKit.lipgloss('red',0.5);

//amount = 0 to 1
